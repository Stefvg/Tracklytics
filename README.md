## Installation

Tracklytics is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "Tracklytics"
```

Make sure you define iOS 7.0 in your podfile:

```ruby
platform :ios, '7.0'
```
